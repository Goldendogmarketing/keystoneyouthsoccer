KEYSTONE YOUTH SOCCER - DEPLOYMENT INSTRUCTIONS
================================================

This is a Node.js server application.

REQUIREMENTS:
- Node.js 22.12 or higher
- PostgreSQL database

SETUP STEPS:
1. Upload all files in this folder to your server
2. Copy .env.example to .env and fill in your values:
   - DATABASE_URL: Your PostgreSQL connection string
   - BETTER_AUTH_SECRET: A secure random string (32+ characters)
   - BETTER_AUTH_URL: Your site URL (e.g., https://yourdomain.com)
   - AUTHORIZE_NET_API_LOGIN_ID: Your Authorize.net credentials
   - AUTHORIZE_NET_TRANSACTION_KEY: Your Authorize.net credentials
   - AUTHORIZE_NET_ENVIRONMENT: 'sandbox' or 'production'
   - RESEND_API_KEY: Your Resend email API key
   - MASTER_ADMIN_EMAIL/PASSWORD: Initial admin credentials

3. In cPanel Node.js App Setup:
   - Application root: /path/to/cpanel-files
   - Application startup file: server/index.mjs
   - Node.js version: 22.x or higher

TO START THE SERVER:
   node server/index.mjs

The server runs on port 3000 by default.
Set PORT environment variable to change.

FOLDER STRUCTURE:
- /public    - Static assets (CSS, JS, images)
- /server    - Server application code
- nitro.json - Server configuration
